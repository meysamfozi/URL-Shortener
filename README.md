# URL-Shortener

A simple URL Shortener application using Nodejs, MongoDB, React, and Materialize CSS.

1) How to run the code

- Back end:
  ```bash
  cd server/
  yarn install
  yarn run start
  ```

- Front end:
  ```bash
  cd server/
  yarn install
  yarn run start
  ```

2) Technologies used
  - Expressjs
  - Mongodb
  - Mongoose (MongoDB object modeling tool)
  - Reactjs
  - React-router (Routing library for React)
  - Materialize css
